Segmentation Parameters

Name Description
Hessian Response This parameter is related to source image contrast. For low contrast images, set it to lower values.
GAC Iteration # If detected cone contours are far outside the visible cone boundaries, set this parameter to a lower value.

Keyboard Shortcuts

Key Action Description
Ctrl+O Open Open a Split Image (.tif). You can open multiple images at once.
Ctrl+G Segment Display Cone Segmentation dialog. On this dialog you can adjust segmentation parameters, then chose to segment cones either on the current image, or on all open images.
Ctrl+S Save Save segmentation results of the current image.
Ctrl+A Save All Save segmentation results of all open images.
Ctrl+M Default Switch to “Default” mouse mode. In this mode, you can zoom in and out the image using mouse wheel.
Ctrl+C Draw Switch to “Draw Cone Contours” mouse mode. In this mode, dragging the mouse around a cone will add a contour.
Note: You can also erase individual contours by pressing and holding Ctrl, then clicking on a contour.
Ctrl+E Edit Switch to “Edit Cone Contours” mouse mode. In this mode, clicking on a contour will switch it to the “Edit” mode, where you can adjust the edges by dragging them with a mouse.
Note: You can also erase individual contours by pressing and holding Ctrl, then clicking on a contour.
Ctrl+D Erase M Switch to “Erase Multiple Cone Contours” mouse mode. In this mode, dragging the mouse around an area on the image will erase all contours inside that area.
Ctrl+W Erase S Switch to “Erase Single Cone Contour” mouse mode. In this mode, clicking on a cone contour will erase it.
Ctrl+Z Redo Undo (redo) the last “Draw” or “Erase” operation.
Ctrl+V Voronoi Toggle display of the Voronoi diagram around cone contour centers.
  Clear Clear all history of manual operations. Free up some memory at the expense of undo (redo) operations.
F1 Help Display this help screen.
