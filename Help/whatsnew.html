What's new – Cone Segmentation Utility

Version 1.1.003 (2022-03-11)

  • New tool button: Toggle Voronoi diagram display.

Version 1.1.002 (2022-03-06)

  • Brightness/Contrast changes can be undone with the “Redo” button.

Version 1.1.001 (2022-02-13)

  • Brightness/contrast is now set separately for each image, and it is not carried over to other images.
  • Fixed issues with deleting annotations in Erase M mouse mode at the image borders.
  • Disabled default behavior of keystrokes such as “W”, “R”, “X”, “Y”, etc.
  • After dragging in image files from a directory, the default save directory is changed to the one where the files were dragged in from.

Version 1.1.0 (2021-05-17)

  • Change mouse cursor to OpenHand when Shift is pressed, and to MoveAll when scrolling the image with Shift+Left Button or Middle Button.
  • Up and Down arrows switch between open images (even if the image list does not have focus).
  • The “Segment” dialog displays a list of open images with check boxes, allowing the user to select which images to segment (and replace existing segmentations). Initially only images that don't have any contours on them are checked, but there is a “√” header to check/uncheck all.
  • A “√” symbol is placed next to image name in the list of open images if there is a matching *_contours.csv file in the same directory as the image (*.tif) file. An image may actually have some contours on it, but they are not saved.
  • A bug with broken *_contours.csv format has been fixed.
  • Toggle pixel interpolation for image zoom resampling (Ctrl+I or menu Split → Toggle Interpolation).
  • F2 toggles cone glyph visibility.
  • Fixed Unicode issues under Windows.
